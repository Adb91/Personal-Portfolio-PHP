# Personal Portfolio Website

A professional portfolio website designed for Software Engineers. Built with **PHP, MySQL, and CSS Grid**.

## Features
- **Responsive Design:** Optimized for both Desktop and Mobile views.
- **Interactive Sidebar:** Smooth navigation for mobile users.
- **Dark/Light Mode:** User-friendly theme switching.
- **Database Driven:** Dynamic content loading for projects and user info.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Grid), JavaScript (Vanilla).
- **Backend:** PHP.
- **Database:** MySQL.

## How to Run
1. Clone the repository.
2. Import the `portfolio_db.sql` file into your local phpMyAdmin.
3. Update `db_connect.php` with your local database credentials.
4. Run using XAMPP or any PHP server.
